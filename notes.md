# Request Methods

- `GET`

`localhost:8080/api/contact`

- `POST`

`localhost:8080/api/contact`

{
    "name": "Jorge Gallaga",
    "email": "jorge@example.com",
    "dataProtection": true
}

`localhost:8080/api/contact/1`

{
    "name": "Alberto Arevalo",
    "email": "alberto@mail.com",
    "dataProtection": true
}

- `DELETE`

`localhost:8080/api/contact/1`
